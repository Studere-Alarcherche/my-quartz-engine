# Reading Map

The first public reading path begins with Proust and moves outward only when a secondary source helps clarify a scene.

## Entry Routes

- Memory and signs: taste, involuntary memory, interpretation.
- Music and form: Vinteuil, recurrence, ineffability.
- Image and attention: Elstir, Vermeer, delayed recognition.
- Space and society: rooms, salons, names, atmosphere.

## Use

Choose a route, pair it with a question from [[Question-Gradient|Question Gradient]], then gather only the evidence needed for a small reading packet.
