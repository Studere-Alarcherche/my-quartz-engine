# NotebookLM Evidence Packet Plan

This plan uses NotebookLM as an evidence-checking layer before questions move into public Reading Pages.

## Packet A｜Text, Time, and Signs

Core sources:
- Proust
- Deleuze
- Genette
- Landy
- Watt / Cambridge Companion as bridge

Questions to verify:
- Is memory awakened or constructed under the pressure of signs?
- How does narrative time shape the subject?
- How does Proust turn time into space?
- Why is self-deception epistemological?

Workflow:
- Upload to NotebookLM.
- Ask NotebookLM for chapter-level evidence.
- Extract verified question blocks.
- Move verified blocks back into Core-Question-Collection.md.
- Only then promote selected questions to public Reading Pages.
