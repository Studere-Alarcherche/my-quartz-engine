# Codex Operating Rules

The canonical Codex instruction file lives at the repository root:

- `AGENTS.md`

Codex should read that file before operating on this repository.

## Rules Codex Must Respect

- Keep the homepage at [[index|content/index.md]].
- Keep public course pages flat under `content/`.
- Do not create a folder-first structure.
- Do not delete, rename, or restructure old content without explicit approval.
- Keep public-facing course pages in English.
- Treat the site as reader-facing, not student-facing.
- Preserve Obsidian wiki links.
- Keep Proust at the center.
- Treat Deleuze as a method lens.
- Treat French as a language precision layer.
